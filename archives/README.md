# The Archives

This directory is Citadel's long-term records-and-lineage layer.

Its default internal stewardship role is:

```txt
Stewardess of Archives and Continuity
```

It exists to preserve:
- historical artifacts
- postmortems
- lessons learned
- long-term memory records
- decision lineage
- retired or superseded material retained for custody

It is not a constitutional authority surface.

Live doctrine, active contracts, and current operational law belong in:

```txt
/core
```

For governing rules, see:

```txt
/core/memory/archives-doctrine.md
```
