# Imports Extraction Ledger

This directory is a temporary extraction surface for useful material recovered from:

- `openclaw/`
- `codex/`

These files are not raw migrations.

They are distilled survivals: patterns, rules, and operating ideas that appear worth translating into Citadel-owned doctrine or sanctioned adapter guidance.

## Extraction Standard

Only material that survives hostile review belongs here.

Surviving material should be:

- useful beyond one runtime
- structurally clear
- recoverable without local mythology
- compatible with Citadel authority, verification, and boundary discipline

Material should be rejected when it is:

- credential-bearing
- identity theater
- runtime superstition
- local path fetish
- duplicated noise
- too implementation-specific to survive as doctrine
