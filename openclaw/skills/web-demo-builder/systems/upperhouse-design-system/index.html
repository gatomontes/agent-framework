Design System Showcase

Quiet luxury with editorial calm.

This showcase distills the sampled Upper House interface into a light, spacious hospitality system shaped by elegant display typography, practical body copy, and muted metallic accents.

The result is intentionally restrained. Buttons are treated as slim bordered invitations, not dominant product CTAs, and the overall system relies on spacing, imagery, and typography to carry the premium tone.
